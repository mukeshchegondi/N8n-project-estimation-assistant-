# N8n-project-estimation-assistant
## Description
An automated n8n workflow that estimates project duration, effort, and risk using AI based on project requirements.

## Input
- Project Name
- Project Type
- Features
- Team Size
- Technology Stack
- Complexity
- Deadline

## Output
- Estimated Time
- Estimated Hours
- Risk Level
- Team Recommendation
- Notes

## Core n8n Nodes
- Webhook (Trigger)
- AI Agent
- Respond to Webhook

## Production URL
https://your-n8n-domain/webhook/project-estimation

## Use Cases
- Freelancers
- Project Managers
- Startups
- Automation Systems

## How to Use
1. Import workflow JSON into n8n
2. Activate workflow
3. Send POST request to webhook
4. Receive estimation response
